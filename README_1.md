# 고객 여정 기반 첫 구매 고객 이탈 분석

> **관측기간 내 첫 구매 고객의 이탈은 구매 과정의 마찰보다, 첫 구매 경험의 깊이 부족과 더 밀접하게 연결되어 있었다.**

## 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 분석 목적 | 신규 고객 이탈을 고객 여정 단계별 경험 차이의 관점에서 분석 |
| 데이터 | [eCommerce Events History in Cosmetics Shop](https://www.kaggle.com/datasets/mkechinov/ecommerce-events-history-in-cosmetics-shop) (약 2,070만 건) |
| 분석 기간 | 2019.10 ~ 2020.02 |
| 분석 대상 | 관측기간 내 첫 구매가 확인된 고객 (110,518명) |
| 이탈 정의 | 첫 구매 이후 30일 내 재구매 없으면 이탈 |
| 사용 기술 | Python, pandas, Scipy, Scikit-learn, Matplotlib |

## 분석 구조

```
탐색 → 장바구니 → 결제 → 재구매
 │         │         │        │
가설1      가설4      가설2    가설3
탐색 부담   구매 결정   가격 혜택  상품 다양성
 증가       지연      경험 부족    부족
```

각 여정 단계에서 관측되는 행동 변수를 정의하고, 해당 변수가 첫 구매 이후 30일 내 재구매 여부와 어떤 관계를 가지는지 가설 단위로 검증하였다.

## 가설별 결과 요약

| 가설 | 여정 단계 | 행동 변수 | 결과 | 핵심 발견 |
|---|---|---|---|---|
| 가설1 | 탐색 | 첫 view → 첫 cart 사이 view 횟수 | **기각** | 탐색이 많을수록 오히려 재구매율 증가 (Q1: 11.9% → Q4: 16.4%) |
| 가설2 | 장바구니 | 첫 구매 시 상대적 가격 혜택 경험 여부 | **채택** | 혜택 미경험 시 재구매율 낮음 (13.1% vs 15.9%), 2~3회가 최적 |
| 가설3 | 결제 | 첫 구매일 고유 상품 수 | **채택** | 상품 수 적을수록 재구매율 낮음 (Q1: 8.1% → Q4: 20.3%, **12.2%p 차이**) |
| 가설4 | 재구매 | 첫 cart → 첫 purchase 소요 시간 | **기각** | 고민 시간 길수록 오히려 재구매율 증가, 3일 이후 정체 |

## 핵심 인사이트

- 초기 가정("여정 단계별 마찰이 이탈을 유발한다")과 달리, **탐색과 결제 과정에서의 시간 소요는 마찰이 아닌 고객 관여도의 지표**로 나타남
- 실제 이탈과 연관된 요인은 **가격 혜택 경험 부족**과 **첫 구매 상품 다양성 부족**
- 이탈의 핵심 원인은 **"과정의 마찰"이 아니라 "경험의 부족"**

## 액션플랜

1. **구매 다양성 확대 유도** (가설3): 연관 상품 추천, 번들 할인으로 첫 구매 상품 수 확대
2. **초기 가격 혜택 경험 설계** (가설2): 신규 고객 대상 초기 2~3회 할인 경험 제공
3. **빠른 구매 고객 사후 관여도 강화** (가설1, 4): 탐색·결제가 빠른 고객에게 사후 가이드 제공

## 분석 한계

- 관찰 데이터 기반이므로 인과관계를 단정할 수 없으며, A/B 테스트를 통한 추가 검증이 필요
- "첫 구매 고객"은 전체 서비스 기준 신규 고객이 아닌, 관측기간 내 첫 구매가 확인된 고객
- 가격 혜택 변수는 정가 정보 부재로 인해 상품 평균 거래가격 기준의 proxy 변수로 정의
